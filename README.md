# 🎓 StudyAI - Your Personal AI Study Assistant

Transform PDFs, videos, and text into comprehensive study notes, quizzes, and flashcards in just 10-30 seconds!

## ✨ Features

- 📝 **AI Study Notes** - Generate comprehensive, organized notes from PDFs, videos, and text
- 🧠 **Smart Quizzes** - Auto-generate multiple-choice quizzes to test your knowledge
- 🎴 **Flashcards** - Create digital and printable flashcards for effective memorization
- 🎯 **Mind Maps** - Visual learning with AI-generated mind maps
- ⚡ **Lightning Fast** - Results in 10-30 seconds
- 💰 **Ultra-Affordable** - Just $0.99/month (₹99 in India)

## 🚀 Getting Started

### Prerequisites

- Node.js 20+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/SultanAiMaster/studyai.git
cd studyai

# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

### Build for Production

```bash
npm run build
npm start
```

## 🏗️ Tech Stack

### Frontend
- **Framework:** Next.js 14.2 (App Router)
- **Language:** TypeScript 5.3
- **Styling:** TailwindCSS 3.4
- **State:** Zustand 4.5
- **Forms:** React Hook Form + Zod

### Backend (Coming Soon)
- **Runtime:** Node.js 20+
- **Framework:** Express 5.0
- **Database:** MongoDB Atlas
- **Cache:** Redis
- **AI:** OpenAI GPT-4 + Whisper
